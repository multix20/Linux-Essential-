> *"La certificación MÁS valorada para SysAdmin Linux. 100% práctica, reconocimiento internacional."*

- Ruta auto-estudio: $500 USD (solo examen)

- **ROI:** Se recupera en 1-3 meses de trabajo

**Temas clave:**
- User & group management
- File permissions & SELinux
- Systemd services
- Firewalld & networking
- Storage management
- Logs & troubleshooting

#### ☁️ AWS Cloud Practitioner

- **Costo:** ~$100 USD
- **Preparación:** AWS Skill Builder (gratis)
- **Práctica:** Free Tier
- **Validez:** 3 años

## 💡 Lecciones Aprendidas

### 🎯 De mis 18 años usando Linux

**Lo que ya domino:**
- Terminal y bash básico
- Troubleshooting y resolución de problemas
- Instalación y configuración de sistemas
- Gestión de archivos y permisos básicos
- Git y control de versiones

**Lo que estoy formalizando:**
- Networking avanzado
- Seguridad y hardening
- Scripting profesional
- Administración de servicios
- Virtualización y containers

### 📚 Del curso NDG Linux Essentials

**Conceptos que transformaron mi perspectiva:**

1. **"Everything is a file"** - La elegancia de la filosofía Unix
2. **Pipes y redirección** - El verdadero poder de la línea de comandos
3. **chmod octal notation** - Finalmente entendí el 755
4. **Systemd** - El nuevo init system que cambió todo
5. **man pages** - Mi nueva mejor amiga para aprender

**Errores que cometía (y ya no):**

| ❌ Antes | ✅ Ahora |
|---------|---------|
| `rm -rf` sin pensar | `rm -i` con alias |
| Trabajar como root | sudo solo cuando necesario |
| No documentar cambios | Git commit detallado |
| Google random solutions | man pages + docs oficiales |
| Scripts sin error handling | try-catch y validaciones |
